Line Following Robot

August 2017

Line Following Robot

  • Programmed (C) a robot to autonomously run motors and encoders based on sensor input
  • Designed and soldered complete circuit board including photo sensors, motors and microprocessor
  • Robot was developed to achieve functionalities such as line following, and detecting light, temperature changes and magnetic fields
  • Designed signal conditioning circuits such as filters and Schmidt triggers using op-amps, thermistors, phototransistors, photodiodes and hall-sensors
